# Task 1 – Data Loading, Cleaning and First Exploration

## Overview

This project focuses on loading, inspecting, cleaning, and performing an initial exploration of a retail sales dataset using Python and pandas. The analysis includes data preprocessing, summary statistics, visualizations, and basic business insights.

## Dataset

* **Dataset:** Sample Sales Data
* **Source:** https://www.kaggle.com/datasets/kyanyoga/sample-sales-data

## Tools and Libraries

* Python
* pandas
* matplotlib

## Tasks Completed

* Loaded the dataset using pandas.
* Inspected dataset structure (shape, columns, data types).
* Converted the `ORDERDATE` column to datetime format.
* Identified and handled missing values.
* Checked for duplicate records.
* Generated summary statistics using `describe()`.
* Analyzed categorical columns using `value_counts()`.
* Created visualizations:

  * Histogram of the `SALES` column.
  * Bar chart showing order count by `YEAR_ID`.
* Exported the cleaned dataset as `cleaned_sales_data.csv`.

## Project Files

* `Week1-Task1.ipynb` – Jupyter Notebook containing all code and comments.
* `note.md` – Data cleaning steps and initial business insights.
* `cleaned_sales_data.csv` – Cleaned dataset.
* `sales_data_sample.csv` – Original dataset.

## Key Business Insights

* The highest number of orders was placed in **2004**.
* Approximately **92.6%** of all orders have the **"Shipped"** status.
* **Classic Cars** is the most ordered product line.
* Most sales are concentrated in lower and middle price ranges, while high-value sales occur less frequently.
