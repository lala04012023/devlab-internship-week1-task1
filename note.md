# Data Cleaning Steps

- Loaded the dataset using pandas.
- Converted the ORDERDATE column to datetime format.
- Filled missing values in ADDRESSLINE2, STATE and POSTALCODE with "Unknown".
- Checked for duplicate records.
- Exported the cleaned dataset as cleaned_sales_data.csv.

# Initial Business Insights

1. The highest number of orders was placed in 2004 (1351 orders).
2. Approximately 92.6% of all orders have the "Shipped" status.
3. Classic Cars is the most ordered product line.
4. Most sales values are concentrated in lower and middle ranges, while high-value sales are relatively rare.
